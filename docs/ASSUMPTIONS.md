# Assumptions

- The user is away and has authorized autonomous decisions within the stated launch goal.
- Approval gates in Superpowers workflows are treated as satisfied by the user's explicit instruction: "approval gate에서 멈추지 말고 ... 계속 진행해."
- The project should prioritize a narrow, immediately useful developer tool over a broad platform.
- A 3-minute quickstart is a hard requirement, so the final project must run with minimal external services and preferably no account setup.
- If GitHub authentication or repository creation fails, the local repository should still be release-ready and the exact blocked command should be recorded.
